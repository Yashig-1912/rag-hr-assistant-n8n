# RAG-Based HR Policy Assistant (n8n)

This project implements a Retrieval-Augmented Generation (RAG) workflow using n8n.

## Features
- Automatically ingests HR policy files from Google Drive
- Chunks and embeds documents into a vector database
- Classifies incoming Gmail queries
- Uses AI agent with RAG to generate policy-based responses
- Sends automated replies via email

## Tech Stack
- n8n
- OpenAI models
- Vector database (Pinecone or similar)
- Google Drive API
- Gmail API
